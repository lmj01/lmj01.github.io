# Font
字体在网页中的应用

去各大网站上看看其他网站默认的字体，一般都是对body样式中存在这样的值

```css 
body {
    height:100%;
 font:400 1em/1.8 'Microsoft Yahei', 'PingFang SC', 'Avenir', 'Segoe UI', 'Hiragino Sans GB', STHeiti, 'Microsoft Sans Serif', 'WenQuanYi Micro Hei', sans-serif
}
```
