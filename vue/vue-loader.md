# Vue-loader

vue 文件称为 SFC(Single File Components)，包含三个template，script，style三个块，也可添加自定义块。
每个快可以通过src属性导入外部文件
vue-loader就是解析这些块的，最后组装成一个ES Module，默认导出的是一个Vue.js组件选项的对象。
